HOW TO RUN:
1. Drag OOP1-Tests.st to the squeak window
2. Select last option in the menu that opened ("filein")
3. Open Tools>Transcript
4. Open Tools>TestRunner
5. Choose "OOP1-Tests" in the left menu in TestRunner
6. Select all the tests you wish to run in the middle menu
7. Choose "Run Selected" in the bottom of the window
8. See results in Transcript

IF YOU WISH TO GENERATE MORE TESTS:
1. Download the Python file
2. Run it
3. OOP1-Tests.st file was generated in the folder
4. Continue as described above
